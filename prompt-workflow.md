# Prompt 流程：小红书本地推 AI 参谋

## 目标

把本地生活商家的零散问题转化成一套可执行输出：

> 店铺信息 -> 套餐诊断 -> 内容生成 -> 核销策略 -> ROI 复盘 -> 下一步建议

## 输入结构

```text
店铺类型：
城市/商圈：
目标用户：
主推产品：
当前套餐：
客单价：
当前困惑：
近 7 天数据：
已有小红书笔记链接或截图：
```

## Step 1：识别经营场景

Prompt:

```text
你是小红书本地生活商家的经营诊断助手。
请根据店铺类型、商圈、目标用户和当前套餐，判断这个商家的主要经营目标。
不要直接生成文案，先判断它是在解决：低成本引流、核销提升、到店加购、复购、还是投放复盘。
输出 3 条判断依据。
```

## Step 2：套餐诊断

Prompt:

```text
请诊断当前套餐是否适合小红书本地生活引流。
重点检查：
1. 是否只是低价，没有清楚的消费场景；
2. 是否有到店加购点；
3. 是否有复购设计；
4. 是否体现门店特色；
5. 是否可能吸引一次性低价用户。
输出：问题、原因、改造方向。
```

## Step 3：内容样片

Prompt:

```text
请基于套餐改造方向，生成一条小红书内容样片。
要求：
- 标题要像真实小红书本地生活笔记；
- 开头 3 秒必须说明用户场景；
- 正文不要夸张承诺；
- 分镜控制在 4 个镜头；
- 结尾必须有评论区互动引导；
- 内容要服务到店、核销和复购，而不是只追求曝光。
```

## Step 4：核销提升策略

Prompt:

```text
请设计到店核销后的二次消费策略。
输出：
1. 店员话术；
2. 加购品建议；
3. 复购券设计；
4. 评论区引导；
5. 哪些做法会伤害用户信任。
```

## Step 5：ROI 复盘

Prompt:

```text
请根据曝光、咨询/收藏、核销、到店加购、复购券领取，判断这次投放下一步该怎么做。
不要只看核销率。
请输出一个复盘表：
- 情况
- 可能原因
- 下一步动作
最后给出：继续投、改套餐、改内容、暂停投放中的一个建议。
```

## 输出原则

- 不承诺爆单。
- 不虚构真实访谈数据。
- 不只给文案，要给经营判断。
- 商家看完后应该知道下一步该改内容、改套餐，还是继续投。
